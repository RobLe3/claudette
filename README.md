# Claudette AI Tools

A collection of AI command-line tools and utilities, separated from claude-flow for independent development.

## Components
- Advanced CLI interface with lazy loading
- Multiple AI backend support (OpenAI, Anthropic, Mistral, Ollama)
- Intelligent caching and performance optimization
- Plugin system for extensibility

## Installation
```bash
pip install claudette-ai-tools
```

## Usage
```bash
claudette "your prompt here"
```

For more information, see the documentation in the docs/ directory.
